Description

History

Editing (YouTube) transcripts by hand can be quite tedious. One common repair is to find sentences, punctuate them, and then capitalize the next sentence. Another is to remove terms like "[Music]" and perform search-and-replace operations for specific patterns.

As a step towards automating this, fixTranscript is an open source tool that lets you load text (or copy-paste), then edit in place, run scripts, and then save the results.

The hope is that it provides an easy framework that you can use and people can extend to perform other, possibly very context-specific tasks.

Next Versions

In the future this tool could do more -- fix urls, load a "change dictionary" to allow custom replacements, include custom scripts, NLP for sentence detection, etc.
