Description

History

FixTranscript was originally designed to make it easier to copy a youTube video transcript and make some simple fixes. The person would take the text from a YouTube transcription and insert periods for the sentence breaks, then use this tool to automatically capitalize the sentences.

See the future section for more.

Thoughts and Possible Future Work

In the future this tool could do more -- fix urls, load a "change dictionary" to allow custom string replacements, even do NLP to find sentences. This tool might also be useful for edits to other text besides YouTube transcriptions.
