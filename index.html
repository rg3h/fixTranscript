FixTranscript
(v1.0.0)
This application helps fix transcripts
  1. go to the YouTube video and copy the transcription (here's how)
  2. paste the transcription text into the box below
  3. set the formatting options and press the Go! button
  4. compare the results
  5. press the copy button
  6. paste the results (ctl-v) into wherever you need them
